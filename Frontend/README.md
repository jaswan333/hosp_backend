# ACE Hospital Management System - React

A complete hospital management system frontend built with React and Vite.

## Features

- **Home Page**: Hero section with stats and quick services
- **Services**: Detailed information about all hospital departments
- **Appointment Booking**: Form with validation and localStorage
- **Appointments History**: Current and completed appointments management
- **Contact & Profile**: Hospital contact info and user profile management
- **About Us**: Comprehensive information about hospital, doctors, and facilities

## Tech Stack

- React 18
- React Router DOM v6
- Vite
- CSS3
- LocalStorage for data persistence

## Installation

```bash
cd hospital-react
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```
